## 网页访问验收

- 本机代理仅监听随机 `127.0.0.1` 端口，未持有随机会话令牌的请求返回 403；
- HTTP/HTTPS 远端连接由对应 Profile 的 SSH `direct-tcpip` 通道发起；SSH 控制连接仍只经用户态 SoftEther 网络栈访问配置的 SSH 目标，未放宽底层隧道拨号限制；
- 远端同源绝对重定向、Cookie Domain/Secure 属性和请求 Origin/Referer 会映射到本机代理源；代理授权 Cookie 不会转发到远端；
- 普通断开会把活动浏览器代理计入引用并提示，强制断开、删除 Profile 和退出程序会关闭代理及空闲连接；
- 当前 Profile 的 DHCP 租约未提供 DNS 和默认网关，已验证直接二层隧道只能访问同子网；浏览器代理改由 SSH 主机解析域名并访问目标；
- 最终实时测试 `TestLiveBrowserResourceThroughIsolatedTunnel` 通过 SoftEther、SSH 跳板和本机代理成功访问 `https://www.baidu.com`，耗时 3.37 秒。

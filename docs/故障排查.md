# 故障排查

| 错误码或现象 | 含义 | 建议 |
|---|---|---|
| `TUNNEL_CERT_UNKNOWN` | 首次连接 SoftEther 服务器 | 与管理员核对证书 SHA-256 指纹后确认 |
| `TUNNEL_CERT_CHANGED` | 已固定的服务器证书发生变化 | 不要绕过；先确认服务器证书是否更换或连接遭到劫持 |
| `TUNNEL_AUTH_FAILED` | SoftEther 用户名或密码错误 | 编辑连接并重新保存隧道密码 |
| `TUNNEL_CONNECT_FAILED` | 无法建立 SoftEther 原生 TLS 会话 | 检查服务器地址、端口（默认 992）、当前网络和防火墙 |
| `TUNNEL_NETWORK_FAILED` | 已认证但 DHCP 或用户态网络初始化失败 | 确认 Virtual Hub 提供 DHCP，并核对 Hub 名称 |
| `TUNNEL_NOT_REQUIRED` | 对“仅 SSH”配置执行了隧道证书或隧道测试操作 | 直接使用“测试 SSH 服务器”或连接按钮 |
| `TUNNEL_TARGET_DENIED` | 底层连接尝试访问配置外的地址 | 两种连接方式都只允许底层拨号到配置的 SSH 主机与端口 |
| `SSH_PORT_UNREACHABLE` | 隔离隧道内或直接网络无法访问 SSH 端口 | 核对连接方式、SSH 地址、端口、DNS 和防火墙；仅 SSH 模式要求当前电脑能直接访问 SSH 端口 |
| `SSH_HOST_KEY_UNKNOWN` | 首次连接 | 与管理员核对 SHA256 指纹后确认 |
| `SSH_HOST_KEY_CHANGED` | 固定指纹发生变化 | 不要绕过；先确认服务器是否重装或遭到劫持 |
| `SSH_AUTH_FAILED` | SSH 认证失败 | 核对用户名及认证方式；密码认证重新保存密码，私钥认证确认服务器已安装对应公钥 |
| `SSH_PRIVATE_KEY_NOT_FOUND` | 未选择私钥、已保存路径失效或文件不可读 | 编辑连接并重新选择私钥文件，确认文件仍存在且当前用户可读 |
| `SSH_PRIVATE_KEY_INVALID` | 私钥格式无效、文件过大或不是普通文件 | 使用有效的 OpenSSH/PEM 私钥文件；不要选择 `.pub` 公钥文件 |
| `SSH_KEY_PASSPHRASE_REQUIRED` | 私钥已加密但未提供口令 | 在连接编辑页填写私钥口令 |
| `SSH_KEY_PASSPHRASE_INVALID` | 私钥口令错误或无法解密 | 重新输入正确口令并测试 SSH 服务器 |
| `BROWSER_RECONNECT_FAILED` | 已打开的网页代理检测到 SSH/隧道断开，但自动恢复失败 | 保持 LabRemote 运行，检查网络和凭据后刷新页面；必要时在客户端重新测试连接 |
| `BROWSER_TARGET_UNREACHABLE` | SSH 服务器无法通过 `direct-tcpip` 访问网页目标 | 核对目标是否应从 SSH 主机访问；访问远端本机服务可填写 `http://127.0.0.1:端口`；确认 SSH 服务允许 TCP 转发 |
| `SFTP_UNAVAILABLE` | SSH 服务未启用 SFTP 子系统 | 请服务器管理员启用 OpenSSH SFTP 子系统 |
| `UPLOAD_CONFLICT` | 远端存在同名文件或文件夹 | 更换目标目录，或确认后勾选覆盖同名文件 |
| `UPLOAD_REMOTE_WRITE_FAILED` | 远端目录不可写、空间不足或连接中断 | 检查远端权限、磁盘空间和 SSH 连接后重试 |
| `UPLOAD_LOCAL_READ_FAILED` / `UPLOAD_LOCAL_CHANGED` | 本地文件无法读取或上传期间发生变化 | 关闭占用程序、确认文件仍存在并重新选择 |
| `UPLOAD_SYMLINK_UNSUPPORTED` | 所选文件夹包含符号链接 | 选择符号链接指向的实际文件或文件夹 |
| `UPLOAD_BUSY` | 同一连接已有上传任务 | 等待当前任务结束或先取消当前上传 |
| `DOWNLOAD_CONFLICT` | 本地存在同名文件或文件夹 | 更换保存目录，或确认后勾选覆盖同名文件 |
| `DOWNLOAD_REMOTE_READ_FAILED` | 远端文件不可读、连接中断或文件发生变化 | 检查远端权限和 SSH 连接；保持断点续传开启后重试 |
| `DOWNLOAD_LOCAL_WRITE_FAILED` | 本地目录不可写、空间不足或文件被占用 | 检查本地权限、磁盘空间和占用程序后重试 |
| `DOWNLOAD_SYMLINK_UNSUPPORTED` | 远端选择中包含符号链接 | 选择符号链接指向的实际文件或文件夹 |
| `DOWNLOAD_LOCAL_NAME_UNSUPPORTED` | 远端名称无法安全保存到 Windows | 在远端将项目改为 Windows 支持的文件名后重试 |
| `DOWNLOAD_BUSY` | 同一连接已有下载任务 | 等待当前任务结束或先取消当前下载 |
| `MCP_UNAUTHORIZED` / HTTP 401 | Token 错误 | 从界面重新复制配置 |
| HTTP 403 | Host、Origin 或来源不是本机 | 使用 `127.0.0.1` 地址，不要通过代理转发 |
| HTTP 429 | 触发速率限制 | 等待一分钟并降低调用频率 |
| `MCP_PROFILE_FORBIDDEN` | Profile 未授权 | 在连接编辑页明确开启 MCP 权限 |

日志位于 `%LOCALAPPDATA%\LabRemote\logs`。日志不包含密码、MCP Token 或默认终端输出；提交问题时可附上相应时间段的 JSONL 日志。

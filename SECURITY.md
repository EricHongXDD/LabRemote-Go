# 安全策略

## 支持版本

| 版本 | 安全更新 |
|---|---|
| 1.0.x | 支持 |
| 更早版本 | 不支持 |

## 报告漏洞

请优先使用仓库 **Security** 页中的私密漏洞报告入口。若该入口暂不可用，请通过仓库所有者的 GitHub 主页联系维护者，且不要在公开 Issue 中披露以下内容：

- 可用的 VPN、SSH 或 MCP 凭据；
- 可复现的认证绕过、命令执行或隧道逃逸细节；
- 私钥、Token、服务器地址或用户数据；
- 尚未修复漏洞的完整利用代码。

报告应包含受影响版本、复现条件、预期与实际行为、影响范围，以及不含真实凭据的最小复现材料。维护者确认问题并准备修复前，请避免公开披露。

## 安全边界

LabRemote 仅承诺将明确配置的 SSH 与网页访问流量送入进程内隔离隧道。Windows 系统 VPN、物理网卡、系统路由以及其他应用流量不属于该隧道。MCP 默认关闭且仅监听回环地址；用户仍需妥善保护导出的 Bearer Token 和本地凭据。
